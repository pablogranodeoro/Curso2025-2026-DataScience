Analysis of the Dataset and Licensing of Data Sources

AccidentesBicicletas_2024

El documento CSV “AccidentesBicicletas_2024” nos proporciona información sobre los accidentes ocurridos en la Comunidad de Madrid con bicicletas involucradas.

VARIABLES DEL DATASET

  • num_expediente: identificador único del expediente administrativo del accidente. Tipo: string. Valores: 2024Sxxxxx (972 únicos en 1.076 registros).
  • fecha: día en que ocurrió el accidente. Tipo: string / date. Valores: fechas distribuidas a lo largo de 2024.
  • hora: hora exacta del accidente. Tipo: string. Valores: 00:00–23:59.
  • localización: lugar del accidente. Tipo: string.
  • número: número de portal o referencia numérica. Tipo: integer.
  • cod_distrito: código del distrito municipal (1-21). Tipo: integer.
  • distrito: nombre del distrito. Tipo: string.
  • tipo_accidente: naturaleza del suceso. Tipo: string. Valores: 12 categorías (caída, colisión, atropello...).
  • estado_meteorológico: condiciones climáticas. Tipo: string. Valores: Despejado, Nublado, Lluvia débil, Lluvia intensa, Se desconoce.
  • tipo_vehiculo: tipo de bicicleta. Tipo: string. Valores: Bicicleta, Bicicleta EPAC.
  • tipo_persona: rol de la persona implicada. Tipo: string. Valores: Conductor, Peatón, Pasajero.
  • rango_edad: intervalo de edad. Tipo: string. Valores: 0-4, 5-9, …, >74.
  • sexo: sexo de la persona. Tipo: string. Valores: Hombre, Mujer, Desconocido.
  • cod_lesividad: código de gravedad o lesión. Tipo: string. Valores: 1-14 (910 válidos, 166 nulos).
  • lesividad: descripción de lesiones. Tipo: string.
  • coordenada_x_utm: coordenada X UTM. Tipo: float. Valores: 431,142 - 454,660.
  • coordenada_y_utm: coordenada Y UTM. Tipo: float. Valores: 4,464,444 - 4,491,676.
  • positiva_alcohol: test de alcoholemia. Tipo: string. Valores: N (no), S (sí).
  • positiva_droga: test de drogas. Tipo: integer. Valores: 1 (positivo).

cabezas_semaforo_bici.csv

El dataset recoge información sobre semáforos con indicaciones para bicicletas.

VARIABLES DEL DATASET

  • tipo_elem: tipo de semáforo (string). Valores: SEMAFORO 2/200 BICI, SEMAFORO 2/200 PEATON-BICI, SEMAFORO 3/100 BICI, SEMAFORO 3/200 BICI.
  • distrito: número de distrito (int).
  • id: identificador único del semáforo (int).
  • id_cruce: identificador del cruce (int).
  • fecha_inst: fecha de instalación (datetime). Valores: 1960-2025.
  • utm_x y utm_y: coordenadas UTM (string, necesitan limpieza).
  • longitud y latitud: coordenadas geográficas (string, necesitan limpieza).

El dataset combina información administrativa con información geográfica, permitiendo análisis estadístico y espacial.

Naming Strategy

  • Dominio Principal: http://movimientosostenible.org
  • Base URI para la ontología: http://movimientosostenible.org/group14#
  • Base URI para los datos instanciados: http://movimientosostenible.org/group14/data/
  • Clases y propiedades (ontología):
    • http://movimientosostenible.org/group14#Bicicleta → ejemplo de clase
    • http://movimientosostenible.org/group14#alcohol → ejemplo de propiedad
  • Patrón para instancias: http://movimientosostenible.org/group14/data/<tipo>/<id>
  • Buenas Prácticas: evitar caracteres castellanos, caracteres especiales, espacios o acentos en URIs.
